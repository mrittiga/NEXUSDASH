# 🚀 NexusDash

**NexusDash** is a modern AI-powered productivity dashboard that centralizes essential tools, insights, and workflows into a clean, intuitive interface. Built with modern web technologies, it delivers a fast, responsive, and scalable experience for developers, students, and professionals.

🌐 **Live Demo:** https://nexusdash-mu.vercel.app

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/nexusdash.git
```

Navigate into the project:

```bash
cd nexusdash
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser:

```
http://localhost:5173
```

---

# 📦 Production Build

Build the project:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
